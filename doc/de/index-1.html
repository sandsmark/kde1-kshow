1. Einführung

kShow ist ein weiterer Bildbetrachter. Er verfolgt ein anderes Konzept als KView. kShow bietet die Möglichkeit durch einfaches anklicken mal schnell ein Bild oder Icon anzuzeigen. Es ist aber auch möglich kShow zu starten und dann ein Verzeichnis zu laden oder peer Drag&Drop zu übergeben. kShow filtert dann alle Dateien ( auch aus Unterverzeichnissen ) mit einem bekannten Bildformat heraus und zeigt sie an.

1.1 Neu

  • Das Configfile hat sich geändert. Dashalb auch die Meldung beim ersten Start.
  • Ein zweites, kShow eigenes Set von Toolbar Icons wird zur Verfügung gestellt
  • Anzeige der Bildernamen und Bildergröße im Vorschaufenster.
  • Positionierung des Vorschaufensters per click auf volle Bildschirmhöhe und an den rechten Bilschirmrand.
  • Kopieren von Bildern per Drag&Drop aus der Bildervorschau in ein Kfm-Fenster oder vergleichbarem hinzugefüght

1.2 Versionen

    V 0.2.0
    • kShow hat ein neues Icon bekommen und Teile des Quellcodes wurden komplett neu geschrieben. deshalb auch der Versionsprung 0.1.4 - 0.2.0
    • Das Durchsuchen von Verzeichnissen kann nun in der Fortschrittanzeige unterbrochen werden.
    • Kopieren von gerade angezeigtem Bild in ein beliebiges Verzeichnis.
    • Die erkannten Bilder können in einer Bildvorschau zur Auswahl angezeigt werden

    V 0.1.4

    • Das Speichern und Laden von Bilderlisten ( Katalogen ) ist nun möglich.
    • Anzeigen weiterer Bildformate.( ILBM / PCX / TIFF / PCD ). Die NetBPM-Tools müssen installiert sein!.

    V 0.1.3

    • Das Help-Button im Toolmenu funktioniert nun auch.
    • Das Laden von Verzeichnissen wird durch eine Fortschrittsanzeige gekennzeichnet.
    • Tastaturfunktionen wurden eingefügt.
    • Bug bereinigt. (Falls auf eine Datei oder ein Verzeichnis keine Zugriffsrechte bestanden beendete sich kShow einfach.)
    • In der Bilderliste Funktionen Anzeigen / Hinzufügen / Entfernen hinzugefügt.

    V 0.1.2 erste öffendliche Version.

    • Hintergrundfarbe frei wählbar.
    • Anzeigen von kompletten Verzeichnissen ( auch Unterverzeichnisse ).
    • Programmfenster wird in der Standarteinstellung der Bildgröße angepaßt.
