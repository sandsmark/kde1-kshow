2. Installation

2.1 Wo bekomme ich kShow?

kShow wird es zum herunterladen als .zip -Datei auf meiner Homepage http://www.fun.horx.de/ralf.berger/kde/ geben. Auf diesen Seiten sind auch Testversionen und anderes zu kShow zu finden.
Auf der Applicationsseite des KDE-Project http://www.kde.org/current.html/ sollte kShow auch zu finden sein.

2.2 Voraussetzungen

Um kShow zu compilieren sollten Sie die neuesten Versionen der Bibliotheken verwenden. Ab der Version 0.2.0 von kShow sind die Qt-Bibliotheken der Version 1.42 oder höher erforderlich. Alle benötigten Bibliotheken finden Sie unter ftp://ftp.kde.org/pub/kde/.

2.3 Compilieren und Installation

Zum Compilieren und zur Installation von kShow geben Sie im Basisverzeichnis von kShow Folgendes ein: 

% ./configure
% make
% make install

Sollten dennoch Probleme auftreten, melden Sie mir diese.
